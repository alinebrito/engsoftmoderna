Informações sobre o Curso

Esta página contém os slides e exercícios usados em duas disciplinas de Engenharia de Software do DCC/UFMG, uma para graduação e outra para alunos de mestrado e doutorado (ambas com 60 horas). O curso trata de princípios e métodos modernos de Engenharia de Software, que são largamente usados por empresas que produzem software com qualidade e produtividade.

O curso possui cinco partes: (1) uma introdução (com uma visão geral do que é Engenharia de Software); (2) Processos de Software (enfatizando métodos ágeis); (3) Projeto de Software; (4) Testes de Software (enfatizando testes de unidades); (5) Manutenção e Evolução de Software (com ênfase em refatoração).

Em breve, pretendeos transformar este material em um livro texto, cujos capítulos devem ser disponibilizados nesta mesma página.
